# Pertemuan 3

## Ekstensi File

Ekstensi file adalah penanda atau akhiran di akhir nama file (biasanya setelah tanda titik, contoh: .docx, .jpg, .pdf) yang menunjukkan **format dan jenis file** tersebut.

- `.md` => Markdown document

- `.py` => File python / source code

## Readme.md

Ekstensi .md = File teks yang memungkinkan formatting.

Dokumentasi = https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax

## Perintah `touch <nama file (termasuk ekstensi)`

Digunakan untuk membuat file melalui terminal

`touch main.txt` tidak sama dengan `touch main.py`

## Python dan Hello World! (print statement)

### Print Statement

Digunakan untuk menampilkan pesan pada *konsol/terminal*

Sintaks = Syntax is the set of rules, principles, and processes that govern the structure of sentences in a given language, specifically word order and phrase arrangement. (Google AI Overview)

Sintaks print:

`print(<pesan>)`

Run file python di konsol:

`python <nama file python>`

Contoh, untuk run file `main.py`:

`python main.py`

Note: Hindari penggunaan spasi dalam penamaan file

hindari: `python file main.py` (tidak boleh)

gunakan **underscore (_)** atau strip (-) untuk mengganti spasi, utamakan underscore: `python file_main.py`

Bahasa pemrograman:

- Python -> Bahasa pemrograman sederhana

- PHP

- Java

- Javascript

- Golang

- Rust

- Dst.

Misal:

- `print('Hello World!)` => Output: Keluar pesan "Hello World!" pada konsol

### Deklarasi variabel

contoh: `a = 3` (mendeklarasikan/mendefinisikan variabel a sebagai 3)

### Comments (komentar)

- Memberi informasi atau niatan digunakannya suatu perintah spesifik

- **Comments tidak dijalankan oleh Python** sehingga umum digunakan untuk *debugging* (memperbaiki program)

- Pemberian komentar pada python dimulai dengan hashtag (#)

- Fungsional hashtag .py berbeda dengan fungsional pada file .md
