Installasi HDFS:
1. Hanya bisa berfungsi pada sistem operasi GNU/Linux (Tidak support untuk OS Windows)
2. Jika menggunakan Windows, maka harus diinstall menggunakan Docker (OS di dalam OS)
3. Docker support OS Windows yang memiliki WSL
4. Installasi Docker
   - Buka https://www.docker.com/
   - Klik "Download Docker Desktop", pilih AMD atau ARM, tergantung spesifikasi
   - Lakukan instalasi
   - Buka aplikasi Docker untuk mengaktifkan aplikasi, kemudian minimize 
   - Run docker-compose.yml dengan command "docker compose up -d" atau "docker-compose up -d" pada direktori yang sama
   - Tunggu beberapa menit hingga docker selesai dijalankan
   - Coba akses di browser http://localhost:9870 (namenode), jika muncul suatu tampilan, maka HDFS sudah berjalan
   - Untuk mematikan container/aplikasi, run "docker compose down" atau "docker-compose down"

Arsitektur HDFS:
1. Misal, terdapat suatu file atau data yang berukuran 100 TB, kita bisa bagi file tersebut ke dalam komputer yang berbeda-berbeda
2. Secara default, masing-masing komputer dapat menyimpan block paling besar 128 MB, sehingga 100 TB ini akan terbagi ke beberapa komputer dengan file sebesar 128 MB masing-masing.
3. Dengan demikian, ada 7813 komputer yang digunakan untuk menyimpan file tersebut
4. Komputer-komputer yang digunakan untuk menyimpan block file ini juga melakukan komputasi. Proses ini dilakukan menggunakan prinsip MapReduce
5. Map: Memproses data di masing-masing block
6. Reduce: Memproses semua hasil perhitungan dari masing-masing block
